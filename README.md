# 🧩 Modern Sudoku Game

An interactive Sudoku puzzle game built with **HTML5, CSS3 (Grid & Flexbox), and JavaScript**. This project features a dynamic puzzle generator, real-time error detection, and a responsive design.

## 🚀 Live Demo
You can play the game here: https://naba79.github.io/Sudoku/

## ✨ Features
* **Dynamic Board Generation:** Every game is unique, generated using a backtracking algorithm.
* **Real-time Validation:** Incorrect entries are highlighted in **red** and cleared automatically to keep the challenge going.
* **Difficulty Scaling:** Pre-configured with a challenging number of empty tiles.
* **Modern UI:** A clean, centered interface with a focus on usability and sleek animations.
* **Responsive Grid:** Built using **CSS Grid** to ensure the board stays perfectly aligned across different screen sizes.

## 🛠️ Built With
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling using Grid, Flexbox, and Inset Box-Shadows for a pixel-perfect layout.
* **JavaScript:** Pure logic for board generation, validation, and game state management.

## 🎮 How to Play
1.  **Select a Tile:** Click on any empty cell on the board.
2.  **Input a Number:** Use the number pad below the board or your keyboard (1-9).
3.  **Avoid Mistakes:** You have a limit of **3 mistakes**. If you enter a wrong number, it will flash red.
4.  **Win the Game:** Fill the entire grid correctly to win!
